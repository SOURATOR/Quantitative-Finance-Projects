## Quantitative Finance Projects
This repository contains a collection of quantitative finance projects exploring various trading strategies, portfolio optimization techniques, and statistical arbitrage methods. The projects utilize historical market data to model risk, identify mean-reverting relationships, and optimize asset allocation.
## Usage

1. Clone this repository.
2. Install dependencies via ```pip install pandas numpy matplotlib seaborn yfinance statsmodels scikit-learn scipy```
3. Launch the Jupyter notebook to view individual projects.
4. For .xlsx files, simply download and open to view the projects.
